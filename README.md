# Thiranex-Customer-Segmentation-Project
# Customer Segmentation using RFM Analysis

## Project Overview
This project analyzes SuperStore order data to segment customers based on Recency, Frequency, and Monetary (RFM) values.

## Key Features
- **Data Cleaning**: Handled UTF-8-sig encoding and normalized date/currency formats.
- **RFM Logic**: Categorized customers into Champions, Loyal, Potential Loyalists, and At-Risk groups.
- **Visualization**: Created a distribution chart of the customer segments.

## Results
- **Loyal Customers**: 325
- **Potential Loyalists**: 220
- **At Risk**: 128
- **Champions**: 122
